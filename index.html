SQL 术语速查

SELECT
指定要查询的列；* 表示所有列。
FROM
指定数据来源表；联表时可用 JOIN 扩展。
WHERE
行级过滤，只保留满足条件的行。
ORDER BY / LIMIT
排序与截断行数，常用于报表与分页。
INSERT / UPDATE / DELETE
增删改：INSERT 用列/值，UPDATE 用 SET + WHERE，DELETE 用 WHERE；无 WHERE 的删改极危险，预览中会提示。
自定义 / 复杂 SQL
可手写 CTE（WITH）、UNION、子查询、多语句、方言专属语法等；不参与可视化校验。
